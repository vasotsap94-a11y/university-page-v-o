# University Web Presence

A modern, responsive university website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **4 Main Pages**:
  - Home: Hero section, featured programs, news & events
  - Academics: Program listings with advanced filtering system
  - Admissions: Application process, tuition info, testimonials
  - Contact: Contact form, department contacts, campus visit information
- **Interactive Filtering**: Search and filter academic programs by degree type, department, and study mode
- **Form Validation**: Client-side validation for contact form
- **Smooth Animations**: Scroll-based animations for enhanced user experience

## File Structure

```
university-page-v/
├── index.html          # Home page
├── academics.html      # Academic programs page with filtering
├── admissions.html     # Admissions information page
├── contact.html        # Contact and visit page
├── styles.css          # All styles with responsive design
├── script.js           # JavaScript for interactivity
└── README.md           # This file
```

## Getting Started

1. Open `index.html` in a web browser to view the website
2. Navigate between pages using the navigation menu
3. On the Academics page, use the filters to search for programs
4. Fill out the contact form on the Contact page

## Customization

- **Colors**: Edit the CSS variables in `styles.css` at `:root` selector
- **Content**: Update text content directly in the HTML files
- **Images**: Replace placeholder gradients with actual images by updating the `.placeholder-image` divs
